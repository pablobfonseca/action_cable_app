# Multi-user Spreadsheet
Using Rails 5 ActionCable and RethinkDB

## Installation
- Install [RethinkDB](https://www.rethinkdb.com/)
- Clone it
- Run `bunde install`
- Initialize rethinkdb with the `rethinkdb` command
- Then `rails server`
